# Casefile Noir (AI Detective RPG)

A top-down detective RPG built with Phaser + Tiled, powered by Gemini.

## Features
- Top-down RPG movement + collisions
- Evidence inspection & notebook
- Suspect interrogation
- AI contradiction detection

## Run locally

### 1) Backend
```bash
cd server
npm install
# add .env with GEMINI_API_KEY
npm run dev
